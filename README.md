# ATmega328 Microcontroller - Personal Learning

This repository contains my personal learning projects, codes, and experiments with the **ATmega328** microcontroller (the same MCU used in Arduino Uno).

---

### 🎯 Goal
Learning microcontroller programming from the ground up — **bare-metal**, no Arduino libraries, just direct register manipulation with AVR-GCC.

### 📁 What's Inside
- Bare-metal C and codes
- Register level programming examples
- Timer, Interrupt, ADC, UART, SPI, I2C projects
- Fuse settings and bootloader related experiments
- Circuit diagrams and documentation
- Personal notes and common mistakes

### 🛠️ Tools & Environment
- **Microcontroller:** ATmega328P
- **Compiler:** avr-gcc + avr-libc
- **Programmer:** USBasp / AVRISP mkII
- **IDE/Editor:** vim
